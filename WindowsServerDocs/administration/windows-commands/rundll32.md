# rundll32

Loads and runs dynamic-link libraries (DLLs). There are no configurable settings for Rundll32. Help information is provided for a specific DLL you run with the **rundll32** command.

## Syntax

```
rundll32 <DLLname>
```

### Parameters

| Parameter | Description |
|--|--|
| [Rundll32 printui.dll,PrintUIEntry](rundll32-printui.md) | Displays the printer user interface. |

## Remarks

Rundll32 can only call functions from a DLL explicitly written to be called by Rundll32.
